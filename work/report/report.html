Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

The browser sends an HTTP-request to a webserver which replies with an HTTP-response. The response is often a new HTML document. The HTTP protocol is the most commonly used data transfer protocol on the internet and works on the application layer. HTTP uses TCP so a handshake is necessary and a connection will be established between the browser and the webserver.

2020-02-17

I dont really know why it would matter for a programming language to be popular. I guess you can assume that it offers something nice if it's gaining popularity, but as long as you can reach the same end result without wasting time, there's really no need to change anything. Im doing my degree project for a company which uses Angular so I'm going to write my degree project in Angular even if it is not as popular as it used to be. I have tried both Angular and Vue and I very much prefer Angular in that case. I chose to compare Angular and Vue at google trends and it shows that Vue has a higher average interest over time but I also noticed that in China Vue has 87% versus Angular's 13%. This country alone changes the world average significantly. In sweden, Angular has a bigger interest than Vue. To recommend a framework to a potential employer I would definitaly need more knowledge than I currently have. But, if the employer would ask me, I'd probably recommend React based on stateofjs.com. I myself have never tried it so the suggestion would be 100% based on this comparison.

Today I Learned

I learned a lot about different frameworks and their popularity. I was accually not expecting Angular to be on the negative side of the opinions line, based on the positive experiences I've had myself.
